## Cíl

Založit nový .NET projekt s markdown-first governance od prvního dne.

## Výstup

- `MetaForge.slnx` — .NET solution v slnx formátu
- `PROPOSALS.md` — master checklist aktivních návrhů
- `Progress.md` — chronologický log realizovaných změn
- `Memories.md` — provozní knowledge file
- `PROPOSALS_NEXT.md` — zásobník kandidátních návrhů
- `README.md` — popis projektu, quick start

## Implementační detaily

- Solution obsahuje složky `/Src/` a `/Tests/`
- Všechny markdown soubory mají definovanou strukturu dle specifikace
- README obsahuje architektonický diagram a rychlý start

## Zpětná vazba / Poznámky

První epic — žádné závislosti. Čistě formátovací a strukturální záležitost.
